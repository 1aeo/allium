## Proposal Template

When adding a new proposal, include:

1. **Executive Summary** - One paragraph overview
2. **Problem Statement** - What issue this solves
3. **Proposed Solution** - Detailed design with visuals
4. **Implementation Plan** - Code changes and timeline
5. **Success Criteria** - How we know it's working
6. **Related Documentation** - Links to relevant files

## Moving Proposals

- **To Planned:** When approved, move to `../planned/` and update status
- **To Implemented:** When complete, move to `../implemented/` with implementation notes
